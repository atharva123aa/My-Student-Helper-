# My-Student-Helper-
I needed a place to track my missed chapters and keep a daily study journal, so I built this small tool. It's basically a simple dashboard for students to keep track of their stuff.
### What it is
It's just a simple HTML/JS page. You can set your class, track "backlogs" (things you forgot to study or missed), and write a little diary entry at the end of the day. Everything saves to your browser so it doesn't disappear when you refresh (which was honestly the hardest part to figure out). 
### Features
- **Study Timer:** A basic 60-minute countdown to keep me focused.
- **Profile:** Pick your class (9th-12th). 
- **Backlog Tracker:** A simple list where I can dump topics I need to catch up on.
- **Daily Journal:** A text box to log how the day went.
  SPECIAL FEATURES
  - **Pomodoro Timer:** Swapped the basic 1-hour timer for a 25m focus / 5m break setup (couldn't figure out a pause button so it just runs lol).
- **Study Plan:** Added a rough 1h 30m schedule block to split my time between hard subjects, pending assignments, and easy revision.
- **Flexbox Layout:** Finally learned how to use flexbox to make a two-column layout so the page isn't just one giant scrolling list.
- **Weekly Calendar:** A customizable Monday-Sunday planner that sits on the side and saves whatever you type straight to your browser.
- **Note of the Day:** An aesthetic sticky note box to drop a quick motivational quote or thought for the day.
- **Custom Fonts:** Imported the `Caveat` cursive font to make the Note of the Day section look like real handwriting.
